# Shortcuts

TissUUmaps contains following keyboard shorcuts:

- `M` - to toggle the markers

- `F` - to expand to fullscreen

- `0` - to hide right menu

- `R` - to toggle regions

- `Escape` - to exit any active region tool


When running TissUUmaps from the graphical interface, you can also use the following shortcuts:

- `Ctrl O` - to open a new image

- `Ctrl S` - to save the current project

- `Ctrl Q` - to quit TissUUmaps
